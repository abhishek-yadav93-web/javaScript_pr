# 🌐 Full Stack Web Application (Authentication + OTP + Simon Game)

It includes:
- User Authentication System 🔐
- Form Data Storage in Database 🗄️
- Password Reset using OTP 📩
- Interactive Simon Game 🎮

---

## 🚀 Features

### 🔹 1. User Authentication
- User Registration & Login
- Secure password handling
- Session-based or token-based authentication

---

### 🔹 2. Form Handling & Database
- User data collected via forms
- Stored in a database (MongoDB / any DB you used)
- Server-side validation implemented

---

### 🔹 3. Forgot Password (OTP System)
- Users can reset their password
- OTP (One-Time Password) is generated
- OTP verification ensures secure password reset

---

### 🔹 4. Simon Game 🎮
- Memory-based game
- User repeats color sequence
- Difficulty increases each level

---

## 🛠️ Technologies Used

### Backend:
- Node.js

### Frontend:
- HTML
- CSS
- JavaScript

### Database:
- MongoDB (or specify your DB)
